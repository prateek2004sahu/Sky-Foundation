# 🌐 Sky Foundation – Admin Portal

## 📌 Overview
The Sky Foundation Admin Portal is a web-based application designed to manage the Universal Skills Passport system. It provides secure admin authentication and a dashboard interface for managing institutional data.

---

## 🚀 Features

- 🔐 Admin Login Authentication
- 🔑 Password Hashing (Security)
- 🤖 CAPTCHA Verification
- 📊 Admin Dashboard
- 🌐 REST API Integration
- ⚡ Fast and lightweight Flask backend

---

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript (Vanilla JS)

### Backend
- Python (Flask)
- Flask SQLAlchemy
- Werkzeug Security

### Database
- SQLite
